Story maps analisis crecimiento urbano en el Distrito central.

Prueba tecnica uso de herramientas para el puesto asesor Técnico I en Información, Comunicación y Tecnología para MEAL (ICT4MEAL)

Se generó esta aplicación web para demostrar el uso y expertis en el uso de tecnologías de la comunicación e información como experto enfocado al uso de las TI geoespaciales tengo sólidos conocimientos en el uso de programas que crean datos usando un componente espacial haciendo más poderoso el análisis de la información para la toma de daciones.

El puesto de asesor técnico especifica que el aspirante debe estar capacitado en el uso de tecnologías de ICT4D, recopilación de datos, gestión de información y sistemas de información geográfica este ejercicio recopila el uso de cada enfoque tecnológico para el desarrollo de una aplicación web que describe un fenómeno físico como es el crecimiento urbano en el distrito central.

Se hace uso de tecnologías como power bi, base de datos postgres SQL, entorno de Arcgis de escritorio y entorno online, google earth engine y lenguaje de programación R tanto para la recolección de datos y análisis de los productos como la representación gráfica. Tengo experiencia en el uso de los requerimientos específicos del puesto de trabajo como recopilación de datos en KOBO y CONMCARE, visualización power bi análisis de datos en lenguaje de programación R, programas estadísticos y sobretodo en GIS donde soy experto calificado como licenciado en las tecnologías de la información geográfica como fotogrametría, catastro, Gestión de riesgo, percepción remota, conocimientos de programación web.
